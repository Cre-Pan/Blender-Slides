# Changelog

## 1.0.0

Initial release for Blender Extensions.

### Added

- Folder-based image import.
- Natural frame sorting for numbered images.
- Layer-oriented plane creation.
- Camera alignment tools.
- Background rendering tools.
- Manual presentation playback handling.

### Changed

- Extension name updated to **Slides Pro** to avoid using protected names in the extension title.
- General project description moved out of version history and into the main description.
- Public support moved to GitHub Issues and Blender community forums.

### Removed

- Startup handler registration. Playback handlers are now registered only when presentation playback is started manually.
